    How AuthentURL Works

    Every URL is scored against 15+ cybersecurity checks, each contributing to a final risk score that determines the classification.

    URL Structure

    Inspects length, dots, subdomains, IP addresses, suspicious symbols, and encoding obfuscation.

    Domain Analysis

    Detects hyphens, brand impersonation of 20+ major brands, suspicious TLDs, and URL shorteners.

    Security Checks

    Verifies HTTPS, validates SSL certificates, checks DNS resolution, and tests domain reachability.

    Domain Age (WHOIS)

    Looks up registration date via WHOIS. Domains under 30 days old are flagged as high risk.

    Keyword Detection

    Scans for 22 phishing-related keywords like "login", "verify", "password", "wallet" in the URL path.

    Risk Scoring

    Each check contributes a weighted score. 0–20 = Safe, 21–50 = Suspicious, 51+ = Phishing.
